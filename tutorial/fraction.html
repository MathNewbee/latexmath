Fraction in LaTeX

1 Command: \frac \cfrac \dfrac \tfrac

A fraction is typeset using \frac,\cfrac,\dfrac or \tfrac. 

                \frac{numerator}{denominator}

There are two differences between these four commands:

1. \tfrac typeset a fraction in text style. \frac, \dfrac, \cfrac typeset a fraction in display style. So a fraction typeset using \tfrac is smaller than others.

2. When typeset a mixed fraction using \frac , like a fraction is embedded within another fraction, the inner fraction will appear smaller. Whereas \tfrac, \dfrac, \cfrac typeset fractions are all the same size.
